數字與符號大冒險

歡迎來到代數維度！這裡的怪物都是帶有數值的「數字魔怪」。你必須使用手中的加、減、乘、除、平方、根號與西格瑪加總武器，將怪物體內的數學代數算至 精準的 0 來消除牠們！

[1] 加法 (+) 讓怪物數字 +1（可用來湊平方數！）
[2] 減法 (-) 讓怪物數字 -1（將數字打到 0 即消滅！）
[3] 乘法 (×) 讓怪物數字翻倍 2 倍
[4] 平方 (²) 讓怪物數字平方（例如 5 變成 25）
[5] 根號 (√) 開根號。大數字怪物的剋星
[6] 西格瑪 (∑) 連射兩隻怪物，將其加總融合為一體！
